io.realm.RealmSchema
^^^^^^^^^^^^^^^^^^^^

.. _io_realm_RealmSchema:

*Superclass:*

- `java.lang.Object <https://docs.oracle.com/javase/7/docs/api/java/lang/Object.html>`__

Class for interacting with the Realm schema. This makes it possible to inspect, add, delete and change the classes in the Realm.**Realm#getSchema() (?)**returns an immutable ``RealmSchema``which can only be used for inspecting. Use **DynamicRealm#getSchema() (?)**to get a mutable schema.



All changes must happen inside a write transaction for the particular Realm.

public abstract **void (?)** remove (`String <https://docs.oracle.com/javase/7/docs/api/java/lang/String.html>`__ className)
