io.realm.rule
=============

If an implementation of this is supplied with the annotation, the :ref:`RunnableBefore.run(RealmConfiguration) <io_realm_rule_RunInLooperThread_RunnableBefore_run_RealmConfiguration_>`  will be executed before the looper thread starts. It is normally for populating the Realm before the test.
