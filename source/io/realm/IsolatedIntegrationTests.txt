io.realm
========

:ref:`io.realm.BaseIntegrationTest <io_realm_BaseIntegrationTest>` 
 | 		`java.lang.Object <https://docs.oracle.com/javase/7/docs/api/java/lang/Object.html>`__ 

Base class for integration tests where each test is run on a separate ROS instance. This adds quite a lot of overhead between each test, so only use this base class if absolutely needed. Otherwise use :ref:`IsolatedIntegrationTests <io_realm_IsolatedIntegrationTests>` .
