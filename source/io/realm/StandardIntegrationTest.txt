Class StandardIntegrationTest
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. _io_realm_StandardIntegrationTest:



io.realm
========

:ref:`io.realm.BaseIntegrationTest <io_realm_BaseIntegrationTest>` 
 | 		`java.lang.Object <https://docs.oracle.com/javase/7/docs/api/java/lang/Object.html>`__ 

The standard base class for integration tests. This class will keep a ROS instance running for all tests to minimize the overhead between each test. NOTE: All tests extending this class should use \`@RunTestInLooperThread\` tests.
