.. _io_realm_internal_Row:



io.realm.internal
=================

Interface for Row objects that act as wrappers around the Realm Core Row object.When the actual class which implements this interface is :ref:`CheckedRow <io_realm_internal_CheckedRow>` , all methods in this interface always validate their parameters and throw an appropriate exception if invalid. For example, methods which accept a column name check the existence of the column and throw `IllegalArgumentException <https://docs.oracle.com/javase/7/docs/api/java/lang/IllegalArgumentException.html>`__  if not found.
