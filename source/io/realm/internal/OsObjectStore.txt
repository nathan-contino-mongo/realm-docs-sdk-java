        |      `Runnable <https://docs.oracle.com/javase/7/docs/api/java/lang/Runnable.html>`__  runnable

   * - Try to grab an exclusive lock on the given Realm file. If the lock can be acquired, the ``runnable``  will be executed while the lock is held. The lock will ensure no one else can read from or write to the Realm file at the same time.
       
       
       
       
       
       
       **Parameters**
       
       - ``configuration`` - to specify the realm path.
          
          
          
       - ``runnable`` - to run with lock.
