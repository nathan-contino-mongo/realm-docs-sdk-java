Class CheckedRow
^^^^^^^^^^^^^^^^

.. _io_realm_internal_CheckedRow:



io.realm.internal
=================

`java.lang.Object <https://docs.oracle.com/javase/7/docs/api/java/lang/Object.html>`__ 
 | 		:ref:`io.realm.internal.UncheckedRow <io_realm_internal_UncheckedRow>` 

Checked wrapper for Row data in Realm Core. All methods called through this will check that input parameters are valid or throw an appropriate exception.For low-level access to a Realm where safety checks were already performed, use :ref:`UncheckedRow <io_realm_internal_UncheckedRow>`  instead for improved performance.
