io.realm.internal
=================

`java.lang.Object <https://docs.oracle.com/javase/7/docs/api/java/lang/Object.html>`__ 

*Implemented interfaces:*



- :ref:`io.realm.OrderedCollectionChangeSet <io_realm_OrderedCollectionChangeSet>` 
- :ref:`io.realm.internal.NativeObject <io_realm_internal_NativeObject>` 


Implementation of :ref:`OrderedCollectionChangeSet <io_realm_OrderedCollectionChangeSet>` . This class holds a pointer to the Object Store's OsCollectionChangeSet and read from it only when needed. Creating an Java object from JNI when the collection notification arrives, is avoided since we also support the collection listeners without a change set parameter, parsing the change set may not be necessary all the time.
