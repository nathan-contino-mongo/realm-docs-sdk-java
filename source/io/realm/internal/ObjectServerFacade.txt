io.realm.internal
=================

`java.lang.Object <https://docs.oracle.com/javase/7/docs/api/java/lang/Object.html>`__ 

Class acting as an mediator between the basic Realm APIs and the Object Server APIs. This breaks the cyclic dependency between ObjectServer and Realm code.
