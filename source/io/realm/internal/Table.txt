io.realm.internal
=================

`java.lang.Object <https://docs.oracle.com/javase/7/docs/api/java/lang/Object.html>`__ 

*Implemented interfaces:*



- :ref:`io.realm.internal.NativeObject <io_realm_internal_NativeObject>` 


This class is a base class for all Realm tables. The class supports all low level methods (define/insert/delete/update) a table has. All the native communications to the Realm C++ library are also handled by this class.
