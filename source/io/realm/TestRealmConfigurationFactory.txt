Class TestRealmConfigurationFactory
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. _io_realm_TestRealmConfigurationFactory:



io.realm
========

**TemporaryFolder (?)**
 | 		`java.lang.Object <https://docs.oracle.com/javase/7/docs/api/java/lang/Object.html>`__ 

Rule that creates the :ref:`RealmConfiguration  <io_realm_RealmConfiguration>`  in a temporary directory and deletes the Realm created with that configuration once the test finishes. Be sure to close all Realm instances before finishing the test. Otherwise :ref:`Realm.deleteRealm(RealmConfiguration) <io_realm_Realm_deleteRealm_RealmConfiguration_>`  will throw an exception in the :ref:`after() <io_realm_TestRealmConfigurationFactory_after__>`  method. The temp directory will be deleted regardless if the :ref:`Realm.deleteRealm(RealmConfiguration) <io_realm_Realm_deleteRealm_RealmConfiguration_>`  fails or not.
