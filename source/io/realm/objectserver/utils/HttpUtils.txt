io.realm.objectserver.utils
===========================

`java.lang.Object <https://docs.oracle.com/javase/7/docs/api/java/lang/Object.html>`__ 

Start and Stop the node server responsible of creating a temp directory & start a sync server on it for each unit test. WARNING: This class is called before Realm is initialized, so RealmLog cannot be used.
